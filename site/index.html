What is OpenClaw?

An open-source AI assistant that runs on your own hardware. It can manage email, calendar, messaging, run automations, do research, and much more ‚Äî all while keeping your data private. Think of it as hiring a personal assistant that lives on your computer.

What hardware do I need?

A Mac Mini is ideal (and required for iMessage integration). You can also run it on a Linux server, VPS, or Raspberry Pi. I can help you source hardware at cost if you don't have anything yet.

How much does it cost to run after setup?

OpenClaw itself is free. You'll pay for the AI model API (like Claude or GPT) ‚Äî typically $10-80/month depending on how much you use it. I'll help you pick the right model and optimize costs.
